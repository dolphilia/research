# research

このリポジトリは様々な分野に関する調査・研究したことをまとめるための場所である。テーマごとにディレクトリが作成され、基本的にMARKDOWN形式でまとめられる。

このリポジトリ内で、調査・研究を元にさらにドキュメントをまとめることもあるが、基本的に草稿・ドラフトまでであり、完成稿を公開する場所としては使用しない。あくまで、調査・研究し、それを元にさらに熟考したり、まとめたりする場所として使用する。

扱うテーマの種類は限定しない。個々のディレクトリが扱うテーマと、別のディレクトリのテーマは、非常に乖離していることがある。よって、ディレクトリ・テーマを横断して扱う必要はなく、一つのディレクトリ内で、一つのテーマに集中して作業することが求められる
